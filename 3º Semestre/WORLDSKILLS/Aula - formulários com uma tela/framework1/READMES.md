## Sobre o projeto

O **Financiamentos2026** é um aplicativo desenvolvido para simular financiamentos de bens, como carros, imóveis ou eletrodomésticos.

A aplicação recebe como entrada:

* Valor do bem
* Número de parcelas
* Taxa de juros mensal
* Taxas adicionais

E retorna:

* Valor das parcelas
* Montante total do financiamento

##  Como executar o projeto

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/Financiamentos2026.git

# Entre na pasta
cd Financiamentos2026

# Instale as dependências
flutter pub get

# Rode o app
flutter run
```
