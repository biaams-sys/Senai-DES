#  Financiamentos2026

---

##  Sobre o projeto

O **Financiamentos2026** é um aplicativo desenvolvido para simular financiamentos de bens, como carros, imóveis e eletrodomésticos.

O usuário informa:

* Valor do financiamento
* Taxa de juros mensal
* Número de parcelas
* Taxas adicionais

E o sistema retorna:

* Valor da parcela
* Valor total a ser pago
